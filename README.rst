Prerequisites
-------------

- python >= 3.6
- flask == 1.1.*
- Flask-WTF == 0.14.*
- tinydb == 3.15.2


Installation
------------

.. code-block:: console

    $ apt install python3 python3-pip
    $ pip3 install -r requirements.txt


Quick Start
-----------

Run the application

.. code-block:: console

    $ flask run

And open it in the browser at http://127.0.0.1:5000/
